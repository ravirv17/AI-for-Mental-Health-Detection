## 📥 How to Download the Model

The trained model weights (500MB+) are too large for GitHub.

1. **Download here:** [Click to Download from Hugging Face](https://huggingface.co/ravirv17/mental-health-detection-weights/blob/main/mental_health_final.pth)
2. **Setup:** Place the downloaded file into the root folder of this project.
3. **Run:** Execute `python app.py` .
